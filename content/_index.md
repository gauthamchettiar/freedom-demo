Freedom is a minimal, flexible Hugo theme built on a component-based architecture. It focuses on simplicity and performance, making it ideal for blogs and documentation sites.

## Key Features

- **Comprehensive Markdown Support**: Fully compatible with all [Markdown syntax](/guide/markdown-syntax) provided by the Hugo renderer.
- **Component Architecture**: Utilize reusable partials and [shortcodes](/guide/custom-shortcodes) for effortless custom layouts.
- **Automatic Dark Mode**: Seamlessly adapts to user light or dark mode preferences.
- **Lightweight & Fast**: Optimized CSS and minimal JavaScript for superior performance.
- **Accessible HTML**: Semantic markup with ARIA labels for improved accessibility.

## Highly Customizable Theming

Override theme colors, fonts, and spacing by customizing CSS variables in your site's `assets/css/theme.css`:

```css {display="assets/css/theme.css"}
:root {
  --color-bg: #fcfcfc;
  --color-text: #222222;

  /* ... */
}

```

- **Dark Mode Support:** Most of the visual element has a dark-mode equivalent configuration. 
- **Typography Support:** Configure font styles for paragraph, heading or code.
- **Spacing & Layout Support:** Configure spacing and layout sizing based on t-shirt sizing. 
- **Dark/Light Chroma Syntax Highlighting Support:** Dual chroma theme for dark/light mode, that can be configured to any style.

Check [Theme Customization](/guide/theme-customization) for a complete list of configuration parameters.
